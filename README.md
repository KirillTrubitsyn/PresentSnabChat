# СнабЧат — Презентация запуска

Презентация AI-ассистента для отдела закупок СГК. 17 слайдов.

## Структура файлов

```
├── index.html      # Основной файл презентации
├── manifest.json   # PWA-манифест
├── sw.js           # Service Worker (офлайн-поддержка)
├── icon.svg        # Иконка (вектор)
├── icon192.png     # Иконка 192×192 (PWA)
├── icon512.png     # Иконка 512×512 (PWA)
└── README.md
```

## Просмотр

**Десктоп:** открыть `index.html` напрямую в браузере или через GitHub Pages.

**Мобильный (PWA):**
1. Открыть ссылку GitHub Pages в Safari (iOS) или Chrome (Android)
2. Safari → кнопка «Поделиться» → «На экран Домой»
3. Chrome → меню ⋮ → «Добавить на главный экран»

Приложение сохранится на главном экране и будет работать офлайн.

## GitHub Pages

В настройках репозитория: **Settings → Pages → Source: Deploy from branch → main → / (root)**.

Сайт будет доступен по адресу: `https://<username>.github.io/<repo-name>/`

## Навигация

- Стрелки ‹ › — переключение слайдов
- Точки внизу — быстрый переход
- Свайп (мобильный) — листание
